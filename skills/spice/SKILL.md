# SPICE Development Skills

This skill package provides automated utilities for SPICE development workflows, focusing on standards enforcement and quality automation.

## Available Skills

This package contains specialized skills for common SPICE development tasks. Each skill is model-invoked automatically when relevant to the current workflow.

## Supporting Documentation

SPICE development standards are documented in `~/.claude/docs/spice/`:

- **CLAUDE-IMPORT.md**: Essential standards for agents and developers
- **SPICE.md**: Complete development standards reference
- **SPICE-Git-Flow.md**: Detailed git workflow and commit standards
- **SPICE-Testing.md**: Testing requirements and patterns
- **SPICE-Worktrees.md**: Worktree management workflows

## Purpose

These skills automate repetitive tasks and enforce standards compliance, ensuring consistent code quality across all SPICE Labs projects.
